# Direct Messaging

Direct Messaging is Studyond's real-time chat system that enables communication between any users on the platform. Each conversation has participants and a timestamped message history.

## Who Uses It

- [[Students]] communicate with [[Experts]] about [[Topics]], application questions, and thesis collaboration details
- [[Students]] coordinate with [[Supervisors]] on research direction, feedback, and academic requirements
- [[Experts]] and [[Supervisors]] can exchange messages to align on [[Thesis Projects]] involving their shared students
- Any user can initiate a conversation with another platform member

## Use Cases

- Coordinating [[Thesis Projects]] between students, company experts, and academic supervisors
- Arranging [[Interview Partners|interviews]] for qualitative research within a thesis
- Clarifying topic requirements before a student applies
- General collaboration and mentoring exchanges
- Follow-up after [[Application Management|application]] submission

## Current Scope

Direct Messaging is a core platform feature alongside [[Application Management]]. It provides the informal communication layer that complements the structured [[Thesis Project Lifecycle]]. Messages are real-time with persistent history across sessions.

## What It Does Not Do

The messaging system does not currently integrate with external tools, does not support group channels beyond direct conversations, and does not have AI-assisted features like message drafting or automated follow-up reminders. These represent potential extensions within the [[Opportunity Space]].
