# Application Management

Application Management is Studyond's lightweight system for handling student applications to thesis topics and job listings. It connects [[Students]] who apply to [[Topics]] with the [[Companies]] that posted them.

## How It Works

When a [[Students|student]] finds a relevant topic through browsing, the [[Matching Engine]], or the [[Student AI Agent]], they submit an application that includes their profile (interests, skills, university, [[Study Programs|study program]]), motivation, and CV. [[Companies]] receive these applications on their topic dashboard and can review candidate profiles, assess fit indicators, and connect with the best-match students.

## Features

- Structured role and skill tagging on applications
- AI-supported candidate matching via the [[Matching Engine]]
- Candidate profiles with university, field, degree level, and skills visible to reviewers
- Application state tracking that feeds into the [[Thesis Project Lifecycle]] (`proposed` -> `applied` -> `agreed`)
- [[Direct Messaging]] for follow-up communication between applicants and company [[Experts]]

## Integration Plans

ATS (Applicant Tracking System) integration is on the roadmap and has been identified as a strategic need for enterprise customers. Enterprise TA teams need native integration rather than manual workflows. Current state: thesis topics operate as a separate workflow managed through the Studyond dashboard. SSO is available at the Enterprise [[Subscription Tiers|tier]].
